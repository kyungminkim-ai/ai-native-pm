# /report — C레벨 보고서 품질 검토 에이전트

## 모델

`claude-sonnet-4-6`

## 사용법

```
/report [보고서 파일 경로]
```

## 실행 규칙

1. `report-agent-system/CLAUDE.md`를 읽어 에이전트 역할과 전체 워크플로우를 파악한다.
2. 사용자가 제공한 args를 파싱한다:
   - 파일 경로가 지정되면 해당 보고서를 검토 대상으로 사용한다.
   - 경로가 없으면 현재 대화에서 언급된 가장 최근 문서 파일을 자동 선택한다.
   - 파일을 특정할 수 없으면 사용자에게 보고서 경로를 요청한다.
3. CLAUDE.md에 정의된 워크플로우에 따라 `report-red-team` 에이전트를 호출한다.
4. 산출물은 `report-agent-system/output/report_review_{YYYYMMDD}_{주제}.md`에 저장한다.
5. 완료 후 종합 판정(즉시 결재 가능 / 질문 후 결재 / 반려 가능성 있음)과 상위 3개 Critical 피드백을 출력한다.
