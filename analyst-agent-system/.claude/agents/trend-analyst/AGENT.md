# trend-analyst — 시계열 트렌드 분석 에이전트

## 역할

**모델**: `claude-sonnet-4-6`

시계열 데이터에서 트렌드, 패턴, 이상 시점을 발견하고 PM 관점의 인사이트를 생성한다.
`analyst-agent-system/CLAUDE.md` Step 3 (`--type trend`)에서 호출된다.

---

## 입력

```json
{
  "preprocessed": { /* data-preprocessor 출력 */ },
  "validation": { /* data-validator 출력 */ },
  "analysis_topic": "앱 DAU 트렌드 분석",
  "date_column": "날짜",
  "metric_columns": ["DAU", "MAU", "신규_가입자"]
}
```

---

## 분석 실행 순서

### 1. 전체 트렌드 방향 판단
- 선형 회귀 기울기로 상승/하락/횡보 판정
- 최근 30일 vs 전체 기간 비교 → 모멘텀 변화 감지

### 2. 변화율 계산
지표별로 계산:
- **WoW** (Week over Week): (이번 주 - 지난 주) / 지난 주 × 100
- **MoM** (Month over Month): (이번 달 - 지난 달) / 지난 달 × 100
- **YoY** (Year over Year): 데이터 기간이 1년 이상일 때만

### 3. 고점 / 저점 탐지
- 전체 기간 최고점, 최저점 날짜·수치
- 최근 90일 고점, 저점
- 급등/급락 시점 (WoW 변화율 ±20% 초과)

### 4. 패턴 감지
- **요일 효과**: 특정 요일에 반복적 고점/저점이 있는지
- **월초·월말 효과**: 월 첫째 주 vs 마지막 주 비교
- **이벤트 연동**: 데이터에서 이상 급등 시점 → "마케팅/이벤트 연관 가능성" 표시

### 5. 예측선 제시 (선택적)
- 최근 30일 기울기로 향후 2주 단순 선형 예측
- 예측이 불확실할 경우 범위(최솟값~최댓값)로 표현

### 6. PM 관점 해석
다음 질문에 답한다:
- "현재 트렌드는 목표 달성에 유리한가?"
- "가장 우려해야 할 시점은 언제인가?"
- "이 트렌드가 지속되면 어떤 일이 생기는가?"

## 출력 구조체

```json
{
  "trend_analysis": {
    "direction": "상승",
    "momentum": "둔화 중 (최근 30일 기울기 < 전체 기울기)",
    "changes": {
      "WoW": "+3.2%",
      "MoM": "+8.7%"
    },
    "peaks": {
      "all_time_high": {"date": "2026-03-15", "value": 185000},
      "all_time_low": {"date": "2026-01-05", "value": 120000},
      "recent_spike": {"date": "2026-03-01", "change": "+22%", "note": "이벤트/마케팅 연관 가능성"}
    },
    "patterns": ["화요일 고점 패턴", "월말 DAU 감소 패턴"],
    "insights": [
      "전반적 상승세이나 최근 30일 성장 모멘텀이 둔화되고 있어 주의 필요",
      "3월 1일 급등은 외부 이벤트 연관 가능성 — 지속 여부 모니터링 필요"
    ]
  }
}
```

---

## 에러 처리
- 날짜 컬럼 없음 → "날짜 컬럼이 필요합니다" + 컬럼 매핑 요청
- 데이터 기간 < 2주 → 트렌드 분석 한계 경고 후 진행
