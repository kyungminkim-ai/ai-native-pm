# 무신사 앱푸시 브랜드 가이드라인

## 무신사 V&T (Voice & Tone)

### 브랜드 개성

무신사는 패션에 진심인 사람들을 위한 플랫폼. 트렌드를 이끌되 과장하지 않고,
팩트 기반으로 솔직하게 전달하는 것이 무신사다운 언어입니다.

### 톤 원칙

| 원칙 | 설명 |
|------|------|
| **간결하게** | 핵심 정보만. 불필요한 수식어 제거 |
| **팩트 중심** | 수치·혜택·제품명을 명확하게 |
| **감탄 금지** | `대박!`, `놀라운!`, `엄청난!` 사용 금지 |
| **명령조 지양** | `사세요!`, `놓치지 마세요!` 지양 |
| **자연스럽게** | 브랜드 톤에서 자연스러운 언어. 딱딱한 번역체 금지 |

---

## 제목 (title) 작성 기준

| 항목 | 기준 |
|------|------|
| 길이 | 15~40자 |
| 구조 | 브랜드명/상품명 + 핵심 혜택/이벤트 키워드 |
| 종결 | 명사형 (예: `단독 발매`, `최대 30% 할인`) |
| 금지 | 감탄사, 명령형, 의미 없는 코드 |

**좋은 예시:**
- `나이키 에어 포스 1 단독 발매`
- `코오롱스포츠 안타티카 단독전`
- `[오늘 종료] 무신사 멤버스데이`

**나쁜 예시:**
- `지금 바로 사세요!` (명령조)
- `대박 할인 중!!` (과장)
- `INSALES_2Q_0412` (코드만)

## 본문 (contents) 작성 기준

### V1 BENEFIT — 혜택 강조형

목적: 할인·쿠폰·적립금 등 수치적 혜택을 명확히 전달

```
(광고) {혜택1} + {혜택2} + {혜택3}
수신거부 : 메인 상단 알림 > 설정 > 알림 OFF
```

| 항목 | 기준 |
|------|------|
| 시작 | 반드시 `(광고) ` |
| 길이 | 40~60자 (수신거부 제외) |
| 혜택 연결 | `+`로 연결, 최대 3개 혜택 |
| 수치 | 퍼센트(%), 원 단위 명확히 |

### V2 BRAND — 브랜드 감성형

목적: 브랜드의 이야기·차별성을 감성적으로 전달

```
(광고) {브랜드/제품의 고유성을 서사로 표현}
수신거부 : 메인 상단 알림 > 설정 > 알림 OFF
```

| 항목 | 기준 |
|------|------|
| 시작 | 반드시 `(광고) ` |
| 길이 | 25~45자 (수신거부 제외) |
| 수치 | 최소화 (혜택 나열 금지) |
| 어조 | 감성적, 서사적 |

## 금칙어 목록

| 유형 | 금칙어 |
|------|--------|
| 과장 표현 | 대박, 초특가, 미친, 엄청난, 놀라운, 역대급 |
| 명령조 | ~하세요!, ~마세요!, 지금 바로 |
| 모호한 표현 | 다양한, 많은, 여러, 좋은, 예쁜 (단독 사용 시) |
| 중복 강조 | `!!`, `~~`, 이모지 남발 |

## 브랜드별 한국어 표기 주의 사항

브랜드 ID(`sourceBrandId`)는 영문 코드이므로 `brand_list.csv`에서 한국어명(`brand_nm`)을 반드시 조회하여 사용.

| 주의 케이스 | 올바른 표기 |
|------------|-----------|
| `martinkimofficial` → ❌ 마틴킴오피셜 | ✅ 마틴킴 (`brand_nm` 참조) |
| `redwing` → ❌ 빨간날개 | ✅ 레드윙 (`brand_nm_eng` 참조) |
| `nikekorea` → | `brand_nm` = `나이키` 사용 |

> **브랜드명 번역 금지.** 반드시 `brand_list.csv`의 `brand_nm` 또는 `brand_nm_eng` 사용.
